== Description ==

Astro is a custom WordPress theme for the Rackhost knowledge base.

== Installation ==

1. Install WordPress Advanced Custom Fields and configure the fields according to the docs.
2. In your admin panel, go to Appearance > Themes and click the Add New button.
3. Click Upload Theme and Choose File, then select the astro.zip file. Click Install Now.
4. Click Activate and then configure the theme settings according to the docs.
